Estuary is an ETL data pipeline platform that lets you set up low- and no-code connections in minutes. With Estuary Flow, you can choose from hundreds of in-house connectors, transform and aggregate data, and power truly real-time applications.

## What Makes Estuary *Estuary*? 🌊

* **Low latency**

    Don't wait minutes or hours for your data. Estuary provides millisecond latency to power your real-time use cases. Or set your own interval for predictable batches.

* **Real-time + batch data**

    Streaming and batch data don't need to be diametrically opposed. With Estuary, you can handle both--even in the same pipeline.

* **Optimized connectors**

    Estuary's in-house, pre-built connectors are optimized specifically for the platform they connect with. Besides resulting in super-low latency, you'll also see [lower ingestion costs](https://estuary.dev/success-stories/headset/) when syncing with platforms like BigQuery and Snowflake.

* **Future-proof schemas**

    Schema inference intelligently infers data types based on field usage while schema evolution handles changes (new, updated, and deleted fields) to keep your pipelines running.

* **Flexible [deployment options](https://estuary.dev/deployment-options/)**
    
    Choose from one of Estuary's public data planes or, for enhanced compliance use cases, opt for a private deployment or BYOC (Bring Your Own Cloud) option.

* **Advanced customization**

    Use Estuary's `flowctl` CLI for powerful pipeline control, including implementing data transformations. Choose between standard SQL or type-safe TypeScript.

* **Transparent pricing**
    
    Estuary is committed to an [easy-to-understand pricing model](https://estuary.dev/pricing/#pricing-calculator) to democratize data. Pay based on the volume of data moved and the number of active connector instances, or [try our free plan](https://dashboard.estuary.dev/register).
